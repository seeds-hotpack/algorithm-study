## ✅ Commit & PR Convention

### Pull Request
- `[N주차] 이름` 형식  
  예: `[1주차] 박성열`

### Commit Message
- `이름: 문제 번호 문제 이름` 형식  
  예: `박성열: 1001 A-B`

### File Name
- `이니셜_문제번호.java` 형식  
  예: `SY_1001.java`, `JSH_11720.java`
